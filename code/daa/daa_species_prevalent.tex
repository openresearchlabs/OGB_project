\subsection{Differential abundance
analysis}\label{differential-abundance-analysis}

Analysis information:

\begin{itemize}
\tightlist
\item
  Taxonomic level: species\_prevalent
\end{itemize}

Taxa that are less prevalent than this (in relabundance) have been
agglomerated:

\begin{itemize}
\tightlist
\item
  Detection threshold: 0.1\%
\item
  Prevalence threshold: 0.0714286
\item
  Number of taxonomic groups: 215
\end{itemize}

Number of significant findings within time points.

\begin{itemize}
\tightlist
\item
  Kruskal test, Benjamini-Hochberg FDR adjustment
\end{itemize}

\begin{verbatim}
before  after 
     1     14 
\end{verbatim}

Number of significant findings within treatments:

\begin{itemize}
\tightlist
\item
  Paired Wilcoxon test, Benjamini-Hochberg FDR adjustment
\end{itemize}

\begin{verbatim}
rice  oat 
  14    8 
\end{verbatim}

Let us look more closely to the significant hits. The logFC corresponds
to logFC in CLR (tp2-tp1).
