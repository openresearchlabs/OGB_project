Analysis information:

\begin{itemize}
\tightlist
\item
  Taxonomic level: genus\_prevalent
\end{itemize}

Taxa that are less prevalent than this (in relabundance) have been
agglomerated:

\begin{itemize}
\tightlist
\item
  Detection threshold: 0.1\%
\item
  Prevalence threshold: 10\%
\item
  Number of taxonomic groups: 139
\end{itemize}

Number of significant findings within time points (Kruskal test,
Benjamini-Hochberg FDR adjustment).

\begin{verbatim}
before  after 
     0      5 
\end{verbatim}

Number of significant findings within treatments (Paired Wilcoxon test,
Benjamini-Hochberg FDR adjustment).

\begin{itemize}
\tightlist
\item
  \textbf{Multiple testing correction:} BH
\item
  \textbf{Statistical test:} Paired Wilcoxon
\end{itemize}

\begin{verbatim}
rice-rice  oat-rice   oat-oat  rice-oat 
        2         0        34         1 
\end{verbatim}

Let us look more closely to the significant hits. The logFC corresponds
to logFC in CLR (tp2-tp1).
