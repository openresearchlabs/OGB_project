\section{Alternate analyses}\label{alternate-analyses}

\begin{itemize}
\item
  \textbf{Beta (Estimate)}: Shows the effect size and direction of a
  predictor on the outcome. Positive means an increase, negative a
  decrease.
\item
  \textbf{95\% CI}: Range where the true beta is likely to fall,
  reflecting precision.
\item
  \textbf{p-Value}: Tests if beta differs from zero; p\textless0.05p
  \textless{} 0.05p\textless0.05 suggests significance.
\end{itemize}
